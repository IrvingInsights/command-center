# Requirements

## Core objective
Create a script-driven FreeCAD model environment for evaluating the PeakHinge mechanism interfaces.

## Mechanism objectives
- Validate pivot geometry and clearances
- Validate ridge scissor folding path
- Compare deployed lock concepts
- Clarify cassette hinge and bearing/load-path concept

## Non-goals for this phase
- full architectural shell
- final structural certification
- final manufacturing drawings
- final weatherproofing package

## Primary questions to answer
1. Does the ridge scissor clear through the full folding path without binding?
2. Do lock variants seat cleanly in deployed mode?
3. Is the pivot detail geometrically sensible and repeatable?
4. Is the cassette bearing/load path concept clear and mechanically honest?
